Principe de base

Python est un langage de programmation interprété, multi-paradigme et multiplateformes. Il favorise la programmation impérative, fonctionnelle et orientée objet.

Syntaxe : Les Variables

x = 5 # Entier nom = "HBZ" # Chaîne de caractères pi = 3.14 # Flottant

Syntaxe : Les Conditions

if x > 10: print("Grand") else: print("Petit")

Syntaxe : Les Boucles

for i in range(5): print(i) # Affiche 0, 1, 2, 3, 4
